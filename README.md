# ECOP_DEITaskTeam_Survey_2025
This repository contains the data, analysis code, and supporting materials for the survey titled "Strategies Advancing Diversity, Equity, and Inclusion (DEI)", conducted by the DEI Task Team of the UN Ocean Decade-endorsed Early Career Ocean Professionals (ECOP) Programme.  The goal of the surve was to better understand DEI strategies being implemented across UN Decade Actions.
